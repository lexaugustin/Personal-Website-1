Task List

Task List is a simple web application that can be used to store a list of tasks that a user would like to accomplish. This is an application I made after learning about jQuery. The application consists of a left panel and a right space. The left panel contains a space that the user can use to add new items to the list, and the right space shows the list of the tasks, with the ability to line-cross a task or delete it after it has been completed. The app does not store data permanently because it is only front-end.
